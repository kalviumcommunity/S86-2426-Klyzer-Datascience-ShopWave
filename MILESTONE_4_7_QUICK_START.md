## Key Jupyter Shortcuts

| Action | Shortcut |
|--------|----------|
| Run cell | **Shift + Enter** |
| Save notebook | **Ctrl + S** |
| Insert cell below | **B** (in command mode) |
| Insert cell above | **A** (in command mode) |
| Delete cell | **DD** (in command mode) |
| Command mode | **Esc** |
| Edit mode | **Enter** |

## Stop Jupyter

When done:
- Go to the terminal
- Press **Ctrl + C** twice
- Or close the terminal window
